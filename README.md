# QuizSSM
Web-app per i quiz SSM pubblicati dal Ministero. Include simulazioni d'esame, modalità ripasso errori, statistiche, ricerca libera e comode scorciatoie da tastiera (tasti 1-5 e 0). Grafica responsive con supporto automatico alla modalità scura. Codice generato e ottimizzato con l'ausilio dell'Intelligenza Artificiale.
